| FedDistill | Pure KD           | Jeong et al. (FedGen/FEDDISTILL+) | `conf/algorithm/feddistill.yaml` | [Complete]    | FEDDISTILL+ variant (shares FedAvg weights AND label-wise logits). Client `FedDistillFit` (`core/client_hooks/feddistill.py`) tracks per-class mean logits (LogitTracker, counts init to 1 to stay finite under non-IID) and trains with `CE + reg_alpha*KLDiv(log_softmax(z), softmax(global_logits[y]))`; server `FedDistillHook` averages client logit matrices (weights still FedAvg-aggregated) and rebroadcasts. Logits travel as bytes via FitRes.metrics / FitIns.config. Deviation: per-round tracker (Flower recreates clients each round), not the reference's cross-round cumulative sum. Verified via unit tests + a 2-round `run(cfg)` smoke exercising the reg path. Ref: `references/feddistill/` (FedGen). |
| FedKD      | Hybrid Q+KD       | Wu et al., 2022         | `conf/algorithm/fedkd.yaml`      | [Complete]    | Verified 2-round local dry run simulation. |
| CFD        | Hybrid Q+KD       | Sattler et al., 2022    | `conf/algorithm/cfd.yaml`        | [Complete]    | Paper-faithful port (no `references/cfd/` to diff against, so tested at elevated rigor). Exchanges quantized soft-labels on the shared public proxy, not weights: constrained b-bit uniform quantization on the probability simplex (`core/softlabel_codec.py`, largest-remainder/Hamilton apportionment; b=1 reduces exactly to argmax one-hot) + delta-coding + zlib. Server `CFDHook` holds a persistent `server_model` refined via dual distillation on the aggregated client soft-labels each round (`pre_aggregate_fit` bypasses FedAvg weight-averaging); its predictions become next round's downstream target. Client `CFDFit` fresh-inits each round (only the upstream delta-reference persists, via `client.state`), digests server labels (KL, round >= 2) then trains private CE. Round 1: no downstream broadcast (server model untrained) but clients still upload soft-labels, seeding dual distillation from round 1. zlib substitutes for CABAC/arithmetic coding (documented caveat). Verified via unit tests (`tests/test_cfd.py`) + a 2-round `run(cfg)` smoke + a CPU end-to-end smoke on CIFAR-10. |
